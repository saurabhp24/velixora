🔧 Bugs Found & Fixed

1

pdfjs-dist RenderParameters — invalid canvas property

4 files passed { canvas, canvasContext: ctx, viewport } to page.render(). The pdfjs-dist type definition for RenderParameters only accepts canvasContext (not a standalone canvas key). Fixed in: usePdfThumbnails.ts, DeletePages.tsx, PdfToJpg.tsx, RearrangePages.tsx.

2

Uint8Array<ArrayBufferLike> not assignable to BlobPart

In india.tsx, pdf.save() returns Uint8Array<ArrayBufferLike> which includes SharedArrayBuffer — not assignable to BlobPart. Fixed by casting: new Blob([bytes.buffer as ArrayBuffer], ...).

3

Missing "ai" category in tools.ts

getToolsByCategory("ai") returned empty array — ToolCategory type only had 'pdf' | 'image' | 'archive'. Added 'ai' to the union type and registered all 11 AI tools (PDF Summarizer, Chat with PDF, OCR, Table Extractor, Resume Analyzer, Contract Analyzer, Compressor, Convert & Merge, PPT Generator, AI Translator, India Tools).

4

India tools page — placeholder content

The /tools/india route had no real implementations. Fully built 8 working micro-tools: Aadhaar PDF optimizer, PAN card compressor, Passport photo formatter (35×45mm), Govt form enhancer, WhatsApp PDF optimizer, Exam document scanner, Invoice extractor (AI), and Assignment formatter.

5

AI Translator — placeholder implementation

The /tools/translate route was a placeholder. Fully implemented with: 14-language selector (7 Indian + 7 international), PDF text extraction, OpenAI streaming translation, real-time progress display, and .txt download.
